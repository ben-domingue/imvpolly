# imvpolly
Please note that this depends on the R package at: https://github.com/ben-domingue/irtimv/tree/main/irtimv-helper

- equivalencesim.R: equivalence of grm and dichotomized versions at boundary for a single item.
- example.R: analysis of a single item with a single choice of parameters
- irw.R: empirical analysis
- sim_fullresponsematrix.R: analysis of simulated data as in irw.R (full response matrix for many items rather than a single item)
